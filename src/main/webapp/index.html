Hiba történt! :-(

Szokásos hiba okai

  1. Az alkalmazást egy fejlesztői környezetből inditotta és nem futatta az npm start vagy npm run webpack:build kódot a terminálban.
  2. Hálózati hiba történt a npm install kód futtatása közben.
  3. Olyan Node.js verziót használ ami nem müködik a JHipster-rel: Kérem használojon LTS (long-term support) verziót.

A kliens oldali kód újrafuttatása

Ha gyorsan akarja, futtasa a hogy újraépítsen és futtason mindent

Ha nagyobb kontrolt szeretne, hogy könnyen tudja debugolni a problémákat, akkor kövesse a következő lépéseket:

  1. npm függöségek telepítése a npm install paranccsal.
  2. Hozza létre a kliens a npm run webpack:build vagy npm start paranccsal
  3. Inditsa el a szervert a paranccsal vagy fejleszői környezet segítségével.
